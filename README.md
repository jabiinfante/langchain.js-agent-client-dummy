# Agent Client

Cliente Angular para pruebas de API con agentes LangChain.

**Demo:** https://jabiinfante.github.io/langchain.js-agent-client-dummy/

## Endpoints que utiliza

- **POST** `http://localhost:3000/message` - Enviar mensajes
- **SSE** `http://localhost:3000/stream` - Recibir mensajes (BaseMessage de LangChain)

## Comandos

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm start
```

El servidor arranca en `http://localhost:4200`
